# Learning: Tars Working Style

## Pattern Name

"Just Do It" - Tars Style

## Discovery Date

2026-01-31

## Context

First Oracle setup session - คุณ Tars ตัดบทเมื่อถูกถามคำถามหลายข้อ

## The Pattern

คุณ Tars ชอบให้:
- **ทำเลย** ไม่ต้องขออนุญาตทุกอย่าง
- **ตรงประเด็น** ไม่อ้อมค้อม
- **ตอบสั้น** ให้ข้อมูลที่จำเป็น
- **Action-oriented** ลงมือทำก่อน ค่อยปรับทีหลัง

## Why It Works

- ประหยัดเวลา
- ลด friction ในการทำงาน
- เห็นผลลัพธ์เร็ว
- ปรับแก้จาก real output ดีกว่าวางแผนมาก

## How to Apply

1. เมื่อได้รับ task → ทำเลย
2. ถามเฉพาะสิ่งที่จำเป็นจริงๆ
3. ให้ options พร้อม recommendation
4. ถ้าไม่แน่ใจ → ทำตาม best guess แล้วบอก
